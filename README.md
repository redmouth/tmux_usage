# Tmux usage
In tmux, every terminal is called panel

##### split horizontally
<pre>
1. press Key <b>ctrl</b> + <b>b</b>
2. press Key <b>shift</b> + <b>"</b>
</pre>

##### split vertically
<pre>
1. press Key <b>ctrl</b> + <b>b</b>
2. press Key <b>shift</b> + <b>%</b>
</pre>

##### To move from one pane to another, simply use the prefix followed by the arrow key:
<pre>
1. press Key <b>ctrl</b> + <b>b</b>
2. press <b>[arrow key]</b>
</pre>
