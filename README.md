# Tmux usage
In tmux, every terminal is called panel

##### split horizontally
1. press Key **ctrl** + **b**
2. press Key **shift** + **"**

##### split vertically
1. press Key **ctrl** + **b**
2. press Key **shift** + **%**

##### To move from one pane to another, simply use the prefix followed by the arrow key:
1. press Key **ctrl** + **b**
2. press **[arrow key]**
